# Conformité

Le module conformité de Dastra vous permet d'évaluer, suivre et améliorer votre conformité à différents cadres réglementaires.

Il se compose d'abord d'une bibliothèque qui va contenir les frameworks, exigences, contrôles, tests,  risques et menaces que vous pourrez importer pour ensuite les utiliser dans vos projets de conformité.&#x20;

La seconde brique est la gestion de périmètres : ceux ci vous permettent de définir l'étendue du projet.

Enfin la brique projets vous permet de gérer votre conformité au sein de projets, regroupement fonctionnels permettant l'articulation entre les contrôles et les frameworks.
