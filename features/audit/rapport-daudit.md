# Partager un rapport de questionnaire

{% embed url="https://youtu.be/hCupYPRbf2U" %}

Une fois les réponses obtenues, vous pouvez consulter, valider, publier, revoir, exporter ou archiver votre rapport de questionnaire.

![Exemple de rapport de questionnaire](<../../.gitbook/assets/image-141.png>)

{% hint style="info" %}
Vous pouvez également visualiser une note de questionnaire et de risque, ainsi que générer un plan d'actions automatiquement construit sur la base des réponses fournies par les répondants.
{% endhint %}
