# Sécurité

<figure><img src="../../../.gitbook/assets/image (399).png" alt=""><figcaption><p>L'onglet vous permettant de configurer les règles de filtrage IP</p></figcaption></figure>

Cet onglet vous permet d'accéder aux paramètres de filtrage IP, vous offrant la possibilité de définir des règles pour limiter l'accès à votre Trust center à certaines plages d'IP. Par défaut, sans ajout de filtrage IP de votre part, votre Trust center sera accessible à toute personne disposant du lien d'accès, sous réserve que le Trust center soit activé.
